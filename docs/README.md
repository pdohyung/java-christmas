## 구현할 기능 목록

### 입력

- [x] 예상 방문 날짜를 입력한다.
    - [x] 숫자가 아닌 경우 예외 처리
    - [ ] 방문할 날짜는 1 이상 31 이하의 숫자가 아닌 경우 예외 처리
- [x] 주문할 메뉴와 개수를 입력한다.
    - [x] 고객이 메뉴판에 없는 메뉴를 입력하는 경우 예외 처리
    - [ ] 메뉴의 개수는 1 이상의 숫자가 아닌 경우 예외 처리
    - [ ] 메뉴 형식이 예시와 다른 경우 예외 처리
    - [x] 중복 메뉴를 입력한 경우 예외 처리

### 출력

- [x] 주문한 메뉴 목록을 출력한다.
- [x] 할인 전 총주문 금액을 출력한다.
- [x] 증정 메뉴를 출력한다.
- [x] 혜택 내역을 출력한다.
- [ ] 총혜택 금액을 출력한다.
- [ ] 할인 후 예상 결제 금액을 출력한다.
- [ ] 12월 이벤트 배지를 출력한다.
- [ ] `[ERROR]`로 시작하는 에러 메시지를 출력한다.

### 비즈니스 로직

- [x] 예상 방문 날짜와 주문 메뉴를 입력 받는다.
- [x] 주문한 메뉴 목록을 보여준다.
- [x] 주문 메뉴로 계산한 할인 전 총주문 금액과 증정 메뉴를 보여준다.
- [x] 방문 날짜와 주문 내용으로 총혜택 내역을 구하고 보여준다.
    - [x] 총혜택을 구하는 기능
- [ ] 할인 후 예상 결제 금액과 12월 이벤트 배지를 보여준다.
